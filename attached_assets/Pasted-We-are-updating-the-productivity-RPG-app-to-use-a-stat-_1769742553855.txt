We are updating the productivity RPG app to use a stat-driven session model instead of task-block completion.
Current stack:
React 18 + TypeScript
Vite
Express backend
PostgreSQL (Drizzle ORM)
Stats stored as JSON: { strength, agility, sense, vitality }
🎯 OBJECTIVE
Replace task-block tapping with a Stat Action Panel system.
Users should:
Tap a stat icon (Strength, Agility, Sense, Vitality)
Start a session
Complete the session manually
Earn XP based on scheduled duration
The sectograph remains informational only (not interactive).
1️⃣ STAT ACTION PANEL
When a user clicks a stat icon:
Open a modal panel with:
Stat Name + Icon
Today’s scheduled time block(s) for that stat
Rotating recommendation tip (short, science-based)
Start Session button
If session is active:
Show elapsed time
Show Complete Session button
No task name input. No manual scheduling here.
2️⃣ SESSION LOGIC
When user taps "Start Session":
Store session in state: { stat: "strength", startTime: Date.now(), scheduledDuration: minutesFromSchedule }
When user taps "Complete Session":
Calculate duration (default = scheduled duration)
Allow user to adjust duration before confirming
Full XP allowed even if time differs from scheduled
3️⃣ XP CALCULATION
Base XP = duration in minutes
Multipliers:
Strength: 1.2x
Vitality: 1.1x
Sense: 1.1x
Agility: 1.15x
If session completed within scheduled time window: +10% bonus
Apply XP:
Update stat value
Update overall level XP
Persist in database
4️⃣ SECTOGRAPH CLEANUP
Modify sectograph:
Remove stat numbers
Remove labels inside arcs
Make arcs cleaner and minimal
System tasks should:
Have subtle outer glow
Slightly thicker stroke
Life tasks remain flat color
Sectograph is read-only. Only center button opens schedule editor.
5️⃣ TIPS SYSTEM
Add rotating tips per stat.
Example:
Strength:
“Sleep 7–9 hours for muscle recovery.”
“Hydrate before and after training.”
“Increase volume gradually to avoid injury.”
Sense:
“Meditation improves emotional regulation.”
“Take short breaks every 60–90 minutes.”
Tips rotate daily. Store locally for now (no need DB persistence yet).
6️⃣ UI BEHAVIOR
When session completes:
Animate XP bar smoothly
Show subtle floating +XP text near stat
Slight glow pulse on stat icon
No flashy celebration
Tone: disciplined, calm, focused.
Do NOT redesign the entire app. Keep current theme. Modify existing structure.
Implement cleanly with reusable components.
